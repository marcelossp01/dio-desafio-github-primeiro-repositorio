ATALHOS DO AMBIENTE GIT
ls -a: Pesquisar arquivos ocultos;
mkdir: Para criar uma pasta;
mv: Para mover arquivos – “mv o nome do arquivo ./o nome da pasta”;
cd: Para mostrar todas as pastas;
cd ..: Para voltar uma pasta;
git add: Para adicionar um arquivo no staged;
git commit -m “”: Para adicionar um commit;
echo > README.md: Para subir o arquivo no site GIT HUB;
:coo: Para colocar um emoji no typora;
git config –list: Para verificar como está onfigurado seu usuário no ambiente GIT;
git commit -m “resolve conflito”: Para corrigir o conflito com outro usuário no mesmo arquivo;
git clone: Para clonar um arquivo no ambiente externo para o ambiente interno.

1 ADICIONAR NA AREA STAGED
